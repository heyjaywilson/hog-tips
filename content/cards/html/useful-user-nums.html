Use dotted user numbers to help with grouping. This allows for quicker selection

Example - Truss 1: 1.1 1.2 1.3 - Truss 2: 2.1 2.2 2.3 - Truss 3: 3.1 3.2 3.3 If I type 1, then it will select all of truss 1. If I type 1 [+] 2, then all of truss 1 and 2 will be selected. Then, if I type .2 [+] [enter] 1.2 and 2.2 will be selected.

SFTWR: 3.4.0 and above
